Guía de Buenas Prácticas

Software

  • Mantén tu sistema operativo y aplicaciones actualizados: Las actualizaciones suelen incluir parches de seguridad que corrigen vulnerabilidades.
  • Utiliza software antivirus y antimalware: Estas herramientas ayudan a detectar y eliminar malware.
  • Considera un firewall: Un firewall actúa como una barrera entre tu dispositivo y la red, bloqueando el acceso no autorizado.

    • Correo electrónico y Contraseñas

  • Ten cuidado con los e-mails sospechosos: No abras correos o links de remitentes desconocidos.
  • Habilita filtros de spam: Esto ayuda a reducir los correos no deseado que llega a tu bandeja de entrada.
  • No uses la misma contraseña para diferentes cuentas: Si una contraseña se ve comprometida, las demás también podrían estar en riesgo.

    • Navegación web

  • Evita sitios web con contenido sospechoso o poco confiable. El protocolo HTTPS indica que la conexión es segura.
  • Evita descargar software de sitios web sospechosos o poco fiables.
  • Antes de abrir cualquier archivo, asegúrate de que esté limpio de malware, escanea los archivos descargados con antivirus.
