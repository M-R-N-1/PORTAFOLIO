## Explicar las características fundamentales de JavaScript para el desarrollo web

1. Qué es JavaScript y su rol en el desarrollo de aplicaciones web.

* JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas y dinámicas. Su rol en el desarrollo web es crucial, ya que permite añadir funcionalidades como animaciones, actualizaciones en tiempo real, interacción con formularios y mucho más, sin necesidad de recargar la página.
A medida que el lenguaje evolucionó, los desarrolladores de JavaScript establecieron bibliotecas, marcos y prácticas de programación y comenzaron a utilizarlo fuera de los navegadores web. En la actualidad, puede utilizar JavaScript para el desarrollo tanto del lado del cliente como del lado del servidor.

2. Las ventajas de usar JavaScript para crear aplicaciones interactivas en el navegador.

* Fácil de aprender y utilizar:
La sintaxis de JavaScript se basa en el lenguaje de programación Java y resulta fácil de aprender y codificar. Los desarrolladores utilizan JavaScript en casi cualquier sitio web y aplicación móvil para scripting del lado del cliente. 

* Obtener independencia de plataformas:
Es posible insertarlo en cualquier página web y utilizarlo con muchos otros marcos y lenguajes de desarrollo web. Una vez escrito, puede ejecutar el código en cualquier máquina. De este modo, permite que el desarrollo de aplicaciones sea independiente de la plataforma.

* Reduce la carga del servidor:
Se puede utilizar JavaScript para reducir la carga del servidor y la congestión de la red ya que es posible ejecutar operaciones lógicas y realizar gran parte del trabajo del servidor en el propio cliente.

* Mejora la interfaz de usuario:
Crea sitios web que facilitan la búsqueda y el procesamiento de información compleja. Los desarrolladores la utilizan para ampliar la funcionalidad y la capacidad de lectura, así como para que la interacción del usuario con el sitio web sea más eficaz.

3. Ejemplos de funcionalidades que solo pueden ser posibles gracias al uso de JavaScript

* Manipulación del DOM: JavaScript puede modificar el contenido y la estructura de una página web en tiempo real, permitiendo actualizar elementos específicos sin recargar toda la página. 
 
* Validación de formularios: JavaScript puede verificar la validez de los datos ingresados por el usuario en un formulario antes de enviarlos al servidor, proporcionando retroalimentación inmediata. 

* Aplicaciones web y móviles: Frameworks como React, Angular y Vue.js, construidos sobre JavaScript, permiten crear aplicaciones web complejas y aplicaciones móviles nativas. 

* Backend con Node.js: Node.js permite ejecutar JavaScript en el servidor, lo que facilita la creación de aplicaciones web escalables y en tiempo real, como las de Uber y Netflix. 

## Descripción del  proyecto realizado

El presente proyecto consiste en una página que permite agregar tareas o actividades a una lista ordenada, así como también la alternativa de eliminarlas del listado con un botón específico. Al agregar las tareas se les asigna un identificador aleatorio y la fecha de creación con los objetos Math y Date respectivamente. Las herramientas utiliadas para la creación de este recurso a través de Visual Studio Code son las siguientes:
* HTML para la creación de la estructura lógica del sitio
* CSS para definir la apariencia que no fue realizada con bootstrap y responsividad 
* JavaScript para la interactividad y funcionalidad de los elementos
* Bootstrap para otorgar mejoras de estilo de forma más rápida y eficiente
* Github para administrar las versiones y publicar el resultado final
